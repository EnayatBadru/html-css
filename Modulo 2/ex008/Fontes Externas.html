Nesse site tem mais fontes possíveis para o seu uso.
Baixar as fontes e colocar na pasta onde existe o site Criado.
Depois de Importar a Fonte nas CSS deve usar o comando.

@Font-Face{
Font-Family:'Pode Dar Qualquer Nome Para a Fonte';
Src:url('Nome da Fonte.ott') format ('opentype'), url('Nome da Fonte.ttf') format ('truetype');
}



Bom isso é Uma das regras das CSS por isso deve colocar por fora onde é adicionado os metodos.

Como Usar a Fonte?

Para usar essa fonte você precisa fazer todos os passos a cima
Depois de fazer tudo a cima voce deve ir nas CSS e depois escolhe normalmente a fonte.
Como por Exemplo:



h1{
Font-Family:'Pode Dar Qualquer Nome Para a Fonte';
}



assim se você quiser usar a fonte deve escrever exatamente como, nomeou a sua fonte nas Condicoes das CSS.
Deve usar o nome Exatamente igual Como Vimos no exemplo a cima.

Como Guardar As Fontes?

Geralmente para guardar as suas fontes deve criar uma pasta e colocar as suas fontes.
Depois no url das fontes no comando das CSS onde nos importamos uma fonte que não exista pelo comando.
Font-Face nos podemos escrevcer exatamente assim se a Sua pasta por exmplo estiver gravada com o nome de Fontes.


@Font-Face{
Font-Family:'Pode Dar Qualquer Nome Para a Fonte';
Src:url('Fonte/Nome da Fonte.ott') format ('opentype'), url('Fonte/Nome da Fonte.ttf') format ('truetype');
}


Fazendo isso o seu site vai conseguir achar a sua fonte em uma pasta,
Sabendo que a sua pasta deve estar onde o site esta sendo criado para nao dar qualquer tipo de problema.
