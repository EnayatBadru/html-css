Usando o Id nas CSS


Como Usar o Id nas CSS?

Bom para usar as Id nas CSS primeiro nos devemos configurar a nossa Pág.
no nosso site HTML


Como escrever um Id no HTML?

Para Escrever uma Id no HTML nos primeiro devemos escrever a tag desejada
NO caso de baixo nos vamos fazer um exemplo com o h1.


Tag em HTML

<h1> Nos podemos escrever qualquer coisa que vai servir de título nivel 1 </h1>

<p> Assim o conteúdo que estiver aqui no primeiro h1 nao será alterado
se nao usarmos o id como vimos a cima. </p>

<h1 id="secundario"> O conteúdo que sera modificado usando a id será somente esse no CSS </h1>

<p>Assim o conteúdo que estiver no h1 de baixo o segundo nesse caso é que será modificado se usarmos o comando de ID no CSS
Isso nao significa que o primeiro conteudo nao pode ser alterado. </p>



Tag em CSS

Agora para usar os comando nas CSS nos vamos usar:


body{
background-color:green;
}

h1{
color:cyan;
}

h1#secundario{
text-align:center;
color:black;
}



Esse comando vai fazer com que o segundo h1 seja modificado
ele vai modificar a sua localizacao, isto é, vai colocar o segundo h1 no meio e vai mudar a sua cor para preto.
