    • Pseudo-classes

    As Pseudo-classes eles são representados por dois pontos ":"

    :visited

    Isso faz com que cada link visitado fica de uma determida cor, isto é, link ou qualquer outra palavra.

    :active

    Esse comando faz que cada link seja clicado ele executa uma determinada função.

    Pseudo-elementos

    As Pseudo-elementos eles são representados por dois pontos seguidos "::"

    ::befor

    Antes de todos os "link", isto é, se fosse a::befor a definição seria exatamente essa, antes de todos os links vai ter uma determinda coisa.

    ::after

    Depois de todos os "link", isto é, se fosse a::after a definição seria exatamente essa, depois de todos os links vai ter uma determinda coisa.

    Alguns Exemplos das Pseudo-Classes e Pseudo-Elementos

    Pseudo-Classes

    :visited

    a:visited{
    color:yellow;
    }

    :active

    a:active{
    color:blue;
    }

    Pseudo-Elmentos

    ::befor

    a::befor{
    content:'Escrever Algo';
    }

    ::after

    a::after{
    content:'Escrever Algo';
    }
